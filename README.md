# IHSetHansonKraus1991
Python package to run and calibrate simple Hybrid (cross-shore + longshore) models.

## :house: Local installation
* Using pip:
```bash

pip install git+https://github.com/defreitasL/IHSetHybridModels.git


```

---
## :zap: Main methods

* [hybrids...](./IHSetHybridModels/hybrid_builder.py):
```python
# model's it self
hybrid...(vars)
```
* [cal_Hybrid_2](./IHSetHybridModels/calibration_2.py):
```python
# class that prepare the simulation framework
cal_Hybrid_2(path)
```
* [cal_Hybrid_2](./IHSetHybridModels/assimilation.py):
